PDF Visual Editor - Help & Commands

File Operations

  • Open PDF - File → Open PDF - Open a .pdf or .omar project file
  • Save - File → Save (Ctrl+S) - Save as .omar project (editable format)
  • Save As - File → Save As (Ctrl+Shift+S) - Save project to a new .omar file
  • Export PDF - File → Export PDF - Export final PDF with all modifications
  • Open Recent - File → Open Recent - Quick access to recent files

Editing Operations

  • Insert Text - Insert → Insert Text - Add new text element
  • Insert Image - Insert → Insert Image - Add image from file
  • Edit Text - Double-click on any text element to edit in place
  • Move Elements - Click and drag to move selected items
  • Resize Elements - Drag the blue resize handles on corners
  • Delete Elements - Select and press Delete key
  • Rotate Elements - Select and press Ctrl+R to rotate 90°

Selection & Clipboard

  • Copy - Edit → Copy (Ctrl+C) - Copy selected elements
  • Paste - Edit → Paste (Ctrl+V) - Paste from clipboard
  • Capture Area - Edit → Capture Area (Ctrl+Shift+C) - Draw rectangle to capture region from PDF
  • Multi-Select - Hold Ctrl and click to select multiple items
  • Rubber Band Select - Click and drag on empty area to select multiple items

Undo & Redo

  • Undo - Edit → Undo (Ctrl+Z) - Undo last action
  • Redo - Edit → Redo (Ctrl+Y) - Redo previously undone action
  • History Panel - View → History - Show/hide command history panel

View Controls

  • Zoom In - Hold Ctrl and scroll mouse wheel up
  • Zoom Out - Hold Ctrl and scroll mouse wheel down
  • Theme - View → Theme → Light/Dark - Switch between themes
  • History Panel - View → History - Toggle undo/redo history view

Inspector Panel

  • Visibility - Check/uncheck boxes to show/hide elements
  • Opacity - Click opacity value to reveal slider (0-100%)
  • Background Opacity - Adjust slider at top to change PDF background transparency
  • Duplicate - Right-click → Duplicate - Copy selected items with offset
  • Join into Folder - Right-click → Join into Folder (Ctrl+J) - Group selected items
  • Create Folder - Right-click → Create Folder - Make new organizational folder
  • Delete - Right-click → Delete or press Delete key
  • Drag & Drop - Reorder elements by dragging in the tree
  • Show/Hide All - Buttons to toggle visibility of all elements at once
  • Edit Names - Click element names to rename for better organization

Thumbnail Panel

  • Navigate Pages - Click on any thumbnail to jump to that page
  • Page Order - Drag thumbnails to reorder pages in exported PDF

Canvas Interactions

  • Right-Click Menu - Right-click on canvas for quick actions
  • Alt+Drag - Drag selection with Alt key to export to other applications
  • Capture Mode - Use Ctrl+Shift+C to enter capture mode, draw a rectangle to extract region

Project Files (.omar)

  • Editable Format - .omar files preserve full editability of all elements
  • References PDF - Links to source PDF (app will prompt if moved)
  • Saves Organization - Inspector panel folders and structure are preserved
  • Theme & Settings - Current theme and preferences saved with project

Tips & Tricks

  • Images larger than 500px are automatically scaled down on insert
  • Text editing supports standard text cursor navigation and selection
  • Scene caching limits memory usage (max 5 pages cached)
  • Use Save for .omar projects, Export PDF for final output
  • Dark theme preference persists across sessions
  • Auto-organization groups Text, Images, and Shapes automatically when opening PDFs
